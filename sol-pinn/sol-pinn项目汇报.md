# sol-pinn 项目汇报

## 1. 项目在做什么

`sol-pinn` 是一个面向托卡马克边界等离子体的研究型项目，目标是用 PINN（Physics-Informed Neural Networks，物理信息神经网络）研究一维 SOL（Scrape-Off Layer，刮削层）平行输运与 sheath 边界建模问题。

项目的核心对象是一个简化的一维稳态输运问题：

- 沿磁力线方向求解电子温度输运
- 热导项具有强非线性，形式为 `T^(5/2)`
- 靶板端满足 sheath 热流边界条件
- 在此基础上继续研究参数泛化与反问题求解

项目整体分为三部分：

1. 用传统有限差分方法构建数值基线
2. 用 PINN 逼近同一个物理问题，并与基线对比
3. 在少量观测下反演 sheath 参数 `gamma`

---

## 2. 这个项目想解决什么问题

### 2.1 正问题

项目首先解决的是一维 SOL 平行输运正问题：

- 已知上游温度 `T_up`
- 已知输运参数与 sheath 系数
- 求解沿场线的温度分布 `T(s)` 以及靶板热流

这类问题传统上可以用有限差分、BVP 等数值方法求解，但 PINN 的价值在于：

- 可以把 PDE、边界条件和数据统一写进一个训练框架
- 后续更容易扩展到稀疏观测、参数识别、反问题
- 适合做参数泛化与 surrogate 建模

### 2.2 反问题

项目进一步解决的是 sheath 参数 `gamma` 的反演问题：

- 给定少量温度观测点
- 先训练 PINN 得到温度场 `T(s)`
- 再利用 sheath 边界解析关系诊断 `gamma`

这使项目从“求解一个方程”扩展到“从有限观测恢复物理参数”。

---

## 3. 这个问题为什么重要

SOL 与 divertor/sheath 边界直接关系到：

- 靶板热流负载
- 边界粒子输运
- 偏滤器设计与运行安全
- 更复杂边界等离子体模型的构建

这个项目的重要性主要体现在四点：

1. 它聚焦于聚变边界物理中一个明确且有物理意义的基础问题。
2. 它把 PINN 引入到含非线性热导和 sheath 边界的输运模型中，验证方法可行性。
3. 它不仅做正问题，还探索了参数反演，这比单纯“求解 PDE”更有研究价值。
4. 它为后续扩展到更复杂的 SOL / divertor / detachment 问题提供了起点。

---

## 4. 项目方法与技术路线

### 4.1 物理模型

项目考虑的一维稳态平行输运方程可写为：

```text
d/ds (kappa_parallel * T^(5/2) * dT/ds) + S(s) = 0
```

边界条件包括：

- 上游 Dirichlet 条件：`T(0) = T_up`
- 靶板 sheath 条件：热流与靶板温度满足非线性关系

项目同时采用了简化的压力闭合关系，将密度与温度联系起来。

### 4.2 数值基线

为了保证 PINN 结果可信，项目先构建了传统数值求解器：

- FD 有限差分求解器
- BVP 边值问题求解器

它们用于：

- 生成参考解
- 交叉验证数值正确性
- 为 PINN 精度评估提供 baseline

### 4.3 PINN 建模

PINN 部分主要包括：

- 多层感知机网络
- Fourier feature 编码
- PDE 残差损失
- 上游边界损失
- sheath 边界损失
- Adam + L-BFGS 两阶段训练

此外，项目还探索了：

- causal training
- 参数化 PINN（输入同时包含 `s` 与 `T_up`）
- 反问题中的 PINN 诊断式反演

- 这对快速扫描参数空间、做代理模型很有价值

## 5.4 反问题结果：sheath 参数 gamma 反演

这是项目最有研究亮点的部分之一。

项目采用的思路不是直接联合优化网络和 `gamma`，而是：

1. 用 PDE + 上游边界 + 观测数据训练 PINN
2. 在靶板端提取 `T(L)` 与 `dT/ds(L)`
3. 利用 sheath 解析关系诊断 `gamma`

### PINN 反演与最小二乘反演对比

![pinn vs ls](figures/inverse/pinn_vs_ls.png)

### 噪声敏感性研究

![noise study](figures/inverse/noise_study.png)

根据实验脚本和测试结果，可以总结为：

- 在干净数据、观测点较密时，PINN 对 `gamma` 的恢复精度很高
- 项目测试要求：
  - `10` 个观测点时误差小于 `1%`
  - `20` 个观测点时误差小于 `0.5%`
  - `1%` 噪声下误差小于 `10%`
- 与最小二乘方法相比：
  - 干净数据下两者都可以很好恢复参数
  - 有噪声时最小二乘更稳健
  - PINN 更适合做物理一致性验证和可扩展反问题框架

## 6. 这个项目的主要结论

可以把项目的结论概括成四点：

1. `sol-pinn` 成功搭建了一套“一维 SOL 输运 + sheath 边界 + PINN + 反演”的完整研究原型。
2. 对于一维稳态 SOL 平行输运问题，PINN 可以较好逼近传统 FD 参考解。
3. PINN 不仅能做正问题，还可以扩展到参数泛化和 sheath 参数反演。
4. PINN 反演在干净、较密观测下效果很好，但在稀疏观测和噪声条件下仍有局限。

---

## 7. 项目的价值与局限

### 7.1 项目价值

- 建立了一个清晰的研究链条：`物理建模 -> 数值基线 -> PINN 求解 -> 反问题`
- 对 PINN 在聚变边界问题中的应用给出了可运行、可验证的例子
- 为后续更复杂模型打下基础

### 7.2 当前局限

- 目前还是一维稳态简化模型
- 物理闭合关系较强，离真实 SOL / divertor 复杂耦合还有距离
- 反演结果对噪声和观测覆盖较敏感
- 还不能替代高保真边界等离子体模拟工具

---

## 8. 后续工作建议

下一步可以沿以下方向推进：

1. 从一维稳态推广到更复杂的输运模型
2. 引入更真实的源项、辐射损失或更多物理变量
3. 提升 PINN 在噪声和稀疏数据下的鲁棒性
4. 将该框架与更高保真边界模拟或实验诊断数据连接

---

## 9. 一句话总结

`sol-pinn` 不是一个通用工业求解器，而是一个高质量的研究型原型：它系统地验证了 PINN 在一维 SOL 平行输运与 sheath 参数反演问题上的可行性，并给出了数值基线、PINN 对比和反演实验结果。
