# PINN + 等离子体研究选题建议（非 SOLPS 代理方向）

## 说明

本文档针对"不打算做 SOLPS 代理模型"的同学，提供其他可行的 PINN + 等离子体研究切入点。  
时间：2026-05-20  
背景：研一学生，PINN + 等离子体方向，SOLPS 代理已有其他人在做。

---

## 方案一（最推荐）：PINN for Runaway Electron (RE) 动力学

### 问题背景

托卡马克 disruption 期间会产生大量 runaway electron，对 ITER 第一壁构成严重威胁。RE 的生成、损失和耗散机制仍不完全清楚，且现有模拟代码（如 CODE、LUKE、GO）计算较慢。

### PINN 可以做什么

- 求解 RE 的 1D 动量空间 Fokker-Planck 方程（含电场加速 + 碰撞 + 辐射阻尼 + knock-on 源项）
- 正向求解：给定等离子体参数预测 RE 分布函数和电流演化
- 逆问题：从实验诊断（如 synchrotron 辐射、HXR 测量）反推 RE 分布函数和关键物理参数

### 为什么适合作为切入点

| 优势 | 说明 |
|------|------|
| PDE 明确 | 1D 动量空间 Fokker-Planck 方程形式已知，比 2D 流体 PDE 简单得多 |
| 数据易得 | 可用 CODE 或 GO 代码生成训练数据，甚至纯 PDE 残差训练 |
| 几乎无竞争 | 目前公开文献中几乎没有用 PINN 做 RE 动力学的系统工作 |
| 可扩展 | 从 1D 动量空间 → 1D + 空间耦合 → 逆问题，逐步加深 |
| 高影响 | RE 问题是 ITER 的核心未解决问题之一 |

### 建议的论文题目方向

> Physics-Informed Neural Network for Runaway Electron Dynamics: Forward Modeling and Diagnostic Inversion

### 发表目标

*Nuclear Fusion* 或 *Plasma Physics and Controlled Fusion*

---

## 方案二：PINN for 杂质输运建模

### 问题背景

聚变装置中杂质（尤其是钨）的输运决定了芯部杂质积累、辐射损失和等离子体性能。STRAHL、SANCO、ATOM 等杂质输运代码存在计算代价和参数扫描困难的问题。

### PINN 可以做什么

- 求解 1D/1.5D 杂质输运方程（对流-扩散-电离-辐射）
- 从实验诊断数据反演杂质输运系数（扩散系数 D、对流速度 V），这是传统方法很难做好的逆问题
- 多杂质品种耦合输运的快速预测

### 为什么适合

| 优势 | 说明 |
|------|------|
| PDE 清晰 | 标准的对流-扩散-源项方程，PINN 最擅长的 PDE 类型 |
| 逆问题是公认难点 | 传统方法用最小二乘拟合，PINN 可以天然地同时优化输运系数 |
| 实验数据可获取 | EAST、KSTAR、DIII-D 等装置有大量杂质诊断数据 |
| 与 SOLPS 代理完全不同 | 这是芯部/边缘杂质物理，不是边界流体代理 |

### 风险提示

实验数据的噪声和处理需要一定经验；建议先用模拟数据做通再过渡到实验数据。

## 方案三：PINN for 合成诊断（Synthetic Diagnostic）

### 问题背景

托卡马克诊断信号的解释通常依赖正向模型（如电磁波传播、辐射输运），但这些正向模型计算慢，难以实现实时诊断反演。

### PINN 可以做什么

选择一个具体诊断，用 PINN 构建从等离子体状态到诊断信号的快速正向模型，进而实现逆问题求解。

### 建议优先选择的诊断

- **ECE（电子回旋辐射）**：辐射输运方程，1D 问题，PDE 非常清晰
- **干涉仪/偏振仪**：电磁波在等离子体中的传播，可以用 Helmholtz 方程
- **反射仪**：微波反射测密度剖面，波方程 + 色散关系

### 为什么适合

| 优势 | 说明 |
|------|------|
| 物理方程纯熟 | 诊断的物理模型是已知的，不需要求解复杂的耦合系统 |
| PINN 的独特优势 | 合成诊断 + 逆问题反演，PINN 比传统方法更自然 |
| 实验连接强 | 直接面向实验数据分析，成果容易被诊断组认可 |
| 范围可控 | 只聚焦一个诊断，问题规模小，适合起步 |

### 建议的工具起点

ECE 的辐射输运方程（1D 吸收/再发射）是最简单的选项。

### 发表目标

*Review of Scientific Instruments* (RSI) 或 *Plasma Physics and Controlled Fusion*

## 方案四：PINN for 刮削层/偏滤器区域简化物理

### 问题背景

这不是 SOLPS 代理，而是用 PINN 做简化物理模型的求解。

### PINN 可以做什么

- **两点模型（two-point model）的 PINN 扩展**：经典的 SOL 简化模型，加上更多物理效应（动量损失、辐射等）
- **偏滤器靶板 sheath 边界条件的 PINN 实现**
- **SOL 径向输运的 1D 简化模型**

### 为什么适合

| 优势 | 说明 |
|------|------|
| 物理透明 | 不需要复杂的代码体系 |
| 代码量小 | 可快速出结果 |
| 非常适合练手 | 可以作为第一篇论文的起点 |

### 劣势

论文影响力有限，可能需要加入创新角度（如反问题或不确定性量化）来提升发表潜力。

## 各方案总对比

| 维度 | RE 动力学 | 杂质输运 | 合成诊断 | 简化 SOL 物理 |
|------|-----------|---------|---------|--------------|
| 新颖性 | ★★★★★ | ★★★★ | ★★★★ | ★★★ |
| 可行性 | ★★★★ | ★★★★ | ★★★★★ | ★★★★★ |
| PDE 难度 | 低-中（1D FP） | 低（1D 对流扩散） | 中 | 低 |
| 发表潜力 | *Nucl. Fusion* | *Nucl. Fusion* | *RSI* / *PPCF* | *NME* / *CS* |
| 实验数据可接入 | 较难（需诊断合作） | 中等 | 容易 | 不需要 |
| 与现有工作的区分度 | 极高 | 高 | 高 | 中等 |

---

## 最终建议（按优先序）

1. **Runaway Electron 动力学**——新颖性最高、竞争最小、PDE 难度可控、可扩展性强，是最推荐的首选
2. **杂质输运逆问题**——物理成熟、逆问题是实际需求，适合与实验组合作
3. **合成诊断**——上手最容易、与实验结合最紧、但需要诊断物理背景
4. **简化 SOL 物理**——安全选项，适合快速出第一篇，但影响力有限
